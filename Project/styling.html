After content, like pictures and text, are added to your HTML file, you'll probably want to "style" your page. Styling refers to making your content appear exactly how you want it to appear. Simple examples would be bolding, or underlining text, setting background colors, positioning content exactly where you want it, etc. There are several different ways to do this.

1. Styling within HTML

This can be done by writing the style directly into each element within the HTML file. For example:

               
                <div style="background-color: blue, color: white"> 
                  This will be white text with a blue background 
                </div>

CSS, or cascading style sheets, uses references, in which you define, to "cascade" onto the elements that you want. Therefore, if there is a certain "style" of text you want to use for all the headers on your page, you only have to define this "style" one time, then apply it to all headers on the page. These "definitions" can be made in the HEAD section of your HTML document, or you can store them in a separate .css file.

2. CSS styling in the head of your HTML doc

CSS can be written directly into your HTML file in the HEAD section, and this will cascade down to wherever those styles are called out on the page. For example:

               
              <head>
                <style>
                  div {
                    background-color: blue;
                  }
                </style>
              </head>
              <body>
                <div>
                  This text will have a blue background
                </div>
              </body>

3. Linking your HTML file to a separate CSS file

Alternatively, CSS can be written on a separate "style sheet" and then linked to the HTML document. This is the preferred method for larger projects. CSS has its own format to be used, in which the "style" of element classes are defined. For example:

                  
                      .header1,
                      .header2, 
                      .header3, 
                      .header4 {
                        font-weight: bolder;
                        background-color: gray;
                        color: navy;
                      }

The above statement is defining the font and background color for headers 1, 2, 3, and 4. For example, anytime an element is listed with a class="header1", the text in that element will bear what was defined above.

 
                  
                    <div class="header1">
                      This text will be a bold, navy color, with a gray background.
                    </div>

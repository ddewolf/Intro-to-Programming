An HTML tag is a piece of HTML code that instructs the content within the tag to do something. An element is some kind of content enclosed within an opening tag and a closing tag. Below you will see the demonstration of multiple tags, and how they appear on the web browser

<p> Tags

This demonstrates text within the first p tag

This demonstrates text within a 2nd p tag. Notice that the text within this p tag begins a 2nd paragraph

<strong> Tags

This text is inside a strong tag. It is basically the same thing as a b tag. Notice that this bold text is on a separate line because it is outside of the 2nd p tag, and therefore, not part of the paragraph above.

<form> Tags

This text is inside a form tag.
This text is outside the form tag. You can see that the form tag is "block" because it caused the text outside of it to start a new line.

<br> Tags

This is a line of text.
A br tag causes the text to start a new line, like this one.

<span> Tags

This text is inside of a span tag. This text is outside the span tag. You can see that the span tag is "inline" because it did not cause the text outside of it to start a new line.

<div> Tags

This text is inside of a div tag.
This text is outside of a div tag. You can see that a div tag is "block" becuase it caused the text outside of it to start a new line.

<img> Tags

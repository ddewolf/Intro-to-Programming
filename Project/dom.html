DOM stands for Document Object Model, and it refers to the way in which your HTML code should be structured. HTML code is a series of elements within elements, within elements. For example, you can see in the snippet below how the <title> tag is a subtag within the <head> tag. Furthermore, notice how the <img> tag is inside a <div> tag, which is inside the <body> tag. Notice how anything within the opening and closing tags are indented. This is done to make the code easier to read. If it were not indented, and everything was inline, it would make it very difficult to tell which elements are inside other elements. This is an example of the "tree-like," or DOM structure. 
              
                <!DOCTYPE HTML>
                <html>
                <head>
                  <title>My Webpage</title>
                </head>
                <body>
                  <div>
                    This is the body of my website
                  </div>
                  <div>
                    <img src= "url" alt="My Pic">
                  </div>
                </body>
                </html>

Components of the Web

The diagram below depicts the basic components of the World Wide Web. In order to access the internet, you run a browser on your PC. A browser is a program used to access and display files hosted on the internet. Examples are Google Chrome and Internet Explorer. The internet, depicted by the cloud, is nothing but a giant network of computers. Internet files stored on these computers are retrieved via the HTTP protocol, and can then be viewed on your PC browser.
